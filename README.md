# 简易人脸识别系统

## 项目介绍

这是一个基于Python、OpenCV和Tkinter的简易人脸识别系统，可以用于应对一些python的课程设计项目。该系统包含人脸录入和人脸识别两个核心功能，采用OpenCV提供的内置模型和数据，确保实现简单、代码精简。

## 功能特点

- **人脸录入**：通过摄像头采集用户人脸图像，输入姓名，自动训练识别模型
- **人脸识别**：实时检测摄像头中的人脸，识别并显示用户姓名
- **多用户支持**：支持多个用户的人脸录入和识别
- **数据持久化**：将采集的人脸图像和训练模型保存到本地
- **简洁界面**：使用Tkinter创建直观的用户界面

## 项目结构

```
PythonProject/
├── main.py           # 主界面模块
├── face_lu.py        # 人脸录入模块
├── face_shibie.py    # 人脸识别模块
├── utils.py          # 工具模块
├── img-re/           # 人脸图像存储目录
└── face_model.yml    # 训练模型文件（可自行生成）
```

## 环境要求

- **Python**：3.7及以上
- **依赖库**：
  - OpenCV (cv2)：用于人脸检测和识别
  - NumPy：用于数据处理
  - Tkinter：用于GUI界面（Python标准库）

## 安装步骤

1. **安装Python**：从[Python官网](https://www.python.org/)下载并安装Python 3.7及以上版本

2. **安装依赖库**：
   ```bash
   pip install opencv-python numpy
   ```

3. **克隆项目**：
   ```bash
   git clone <项目地址>
   cd PythonProject
   ```

## 使用方法

1. **运行程序**：
   ```bash
   python main.py
   ```

2. **人脸录入**：
   - 点击"人脸录入"按钮
   - 在弹出的对话框中输入您的姓名
   - 面向摄像头，系统会自动采集5张人脸图像
   - 采集完成后，系统会自动训练模型

3. **人脸识别**：
   - 点击"人脸识别"按钮
   - 面向摄像头，系统会实时检测并显示识别结果
   - 按ESC键退出人脸识别

## 技术实现

- **人脸检测**：使用OpenCV的Haar级联分类器
- **人脸识别**：使用OpenCV的LBPH（Local Binary Patterns Histograms）算法
- **界面设计**：使用Tkinter创建GUI界面
- **数据存储**：使用结构化的文件名存储用户信息

## 注意事项

- 确保摄像头正常工作
- 人脸录入时，确保光线充足，面部清晰可见
- 人脸识别时，保持面部在摄像头视野范围内
